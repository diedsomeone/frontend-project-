About Student Performance Tracker

Student Performance Tracker is a modern web application designed to help colleges, teachers, and students monitor and improve academic performance. The system provides a comprehensive dashboard for each user role, automatic performance analysis, and actionable suggestions for improvement. It is built to be user-friendly, visually appealing, and highly interactive.

Student

  • View marks, attendance, grades, and analysis
  • Get suggestions and advice for improvement
  • See fee updates, news, and important info

Teacher

  • Update attendance and marks
  • View student performance and give feedback

Admin

  • Manage fee/news updates and announcements
  • Manage students and teachers
